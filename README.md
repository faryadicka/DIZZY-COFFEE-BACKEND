## ðŸ“Œ How To Install?

- Clone This Repo

```
git clone https://github.com/faryadicka/dizzy-coffeeshop-be
```

- Go To Folder Repo

```
cd dizzy-coffeeshop-be
```

- Install Module

```
npm install <package-name>
```

- Type `npm run startDev` To Start The Project
